# Employee Performance & Salary Insights
## Project Overview
This analysis focuses on Human Resources data to evaluate how performance scores correlate with salaries and experience across different departments (IT, HR, Finance, Marketing).

## Key Metrics Analyzed
- **Average Salary:** $11,400 per month.
- **Top Performing Department:** Finance (Average Performance: 92%).
- **Salary Range:** From $9,000 (HR) to $15,000 (Finance).
- **Staff Distribution:** IT department represents 40% of the workforce.

## Dashboard Features
- **Departmental Comparison:** Visualizing which departments are the most expensive vs. most productive.
- **Individual Performance Tracking:** Monitoring high-flyers like "Hana" (Top Performance & Salary).
- **KPI Cards:** Displaying Max/Min/Median values for salaries and scores.
